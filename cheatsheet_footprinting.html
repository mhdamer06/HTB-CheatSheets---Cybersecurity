WHOIS / DNS COMMANDS
whois target.com# Registrar, owner, nameservers, creation date
nslookup target.com# Basic DNS A record query
dig target.com ANY# All DNS records
dig @ns1.target.com target.com AXFR# Zone transfer attempt (misconfigured servers)
dig -x 1.2.3.4# Reverse DNS lookup (PTR)
host -t MX target.com# Mail server records
dnsenum --dnsserver ns1.target.com target.com# Automated DNS enumeration
fierce --domain target.com# DNS brute-force + zone transfer
sublist3r -d target.com# Subdomain enumeration via OSINT
